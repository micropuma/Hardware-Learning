# 如何配置虚拟机
## 配置网络以及ssh连接  
1. sudo apt install net-tools
2. sudo apt install openssh-server

## github配置
1. ssh-keygen
2. vim ~/.ssh/id_rsa.pub，并拷贝到github的settings中
3. git config --global user.name "micropuma"
&& git config --global user.email "micropuma@163.com"
